Release timeline — newest first

13 Jun 2026 Beta 1.3.3

The Interaction Window — screens that park, not die.

Generated interactive screens used to vanish after ~90 seconds. Now they stay live while the agent's engaged, then quietly park — tap a parked screen to wake the agent for that one tap, with nothing running in the background.

Neural Interface Generator

  • Park is free — the screen and your taps stay put when the live window closes
  • Tap-to-wake — costs a turn only when you actually do something
  • Nothing lost — an unreachable tap rides along with your next message

Presence marker

  • ● listening · ◌ working · ○ parked — shows what the screen is doing
  • Breadcrumb title replaces the unclear ↑ arrow
  • iOS first-connection lockup fixed — same cause as 1.3.2's boot freeze
11 Jun 2026 Beta 1.3.2

iOS boot fix and background session titling.

The headline: iOS first-start no longer freezes — the webview choked on a timer poll loop, now event-driven and verified on hardware. Auto-titling also moved out of your conversation into a background distiller.

Session Titles

  • A background agent names sessions after the turn — leaner main prompt
  • Optional cheap sessionTitleModel; manual renames always win

Hardening

  • Glasses interactive UI — seven navigation bugs fixed
  • 1008 reconnect-storm fixed; Docker/VPS installs self-diagnose
09 Jun 2026 Beta 1.3.1

Connection and onboarding hardening.

Mostly fixes, not features. A reported lockout left the app stuck on "Plugin version unknown" with a dead retry — the relay editor is now always reachable when you're disconnected, so a stale address can be fixed in place.

Connection

  • New Plugin not verified state stops the misleading "too old" error
  • A stale banner no longer hides a real bad-token / unreachable failure

Setup & safety

  • Clean install — no --dangerously-force-unsafe-install needed
  • Windows reserved-port fix; safe relay port on fresh installs
09 Jun 2026 Beta 1.3.0

Neural Modules — the agent builds the UI.

The big one. When a custom interface beats a text reply, the agent renders interactive tiered lists, titles and text on the glasses — ask "what movie times this afternoon?" and pick from a scrollable list with your ring. Plus a whole new streaming and messaging stack.

Neural Modules

  • Neural Interface Engine v1 — interactive, ring-driven screens
  • Emoji Reactor & Pace Modulator — agent reacts & paces mid-stream
  • Topic Distiller — names sessions from context

Streaming & messaging

  • Continuous mode — teletype line-by-line scroll
  • Dynamic flow — eased reveal, paced paragraph hand-offs
  • Phone session switcher with deep search; new app theme & icons
01 Jun 2026 Announcement

Working with Pekel Labs.

Exploring new methods for interfacing with OcuClaw. New hardware is in testing that should complement the R1 — particularly suited to an upcoming feature beta testers will get to play with first.

03 Apr 2026 v1.2.4 · Launch

Even Hub Launch — OcuClaw on Even G2.

Where it started. Bidirectional chat with your OpenClaw agents on the glasses, real-time streaming, image attachments, full session history, voice, and a responsive phone companion across 22 dedicated glasses screens.

Messaging & voice

  • Streaming chat, image attachments, slash-command forwarding
  • Soniox STT — tap & menu modes, real-time transcription overlay

Agent & Even AI

  • Dedicated Even AI routing — custom prompts, models, defaults
  • Dynamic model catalog, per-session thinking, skills browser
